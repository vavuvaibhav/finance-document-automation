# Invoice to Purchase Order: AI-Driven Document Automation

## 📌 Project Overview
In modern corporate finance and supply chain operations, manual data entry from financial documents is time-consuming and prone to human error. This project demonstrates an automated workflow designed to extract critical information from invoices and streamline the reconciliation process with Purchase Orders (POs). 

Using an AI-powered data extraction platform (Nanonets), this agent automates the ingestion of unstructured invoice documents, extracts key structured fields, and maps them efficiently for financial reporting and processing.

## 🚀 Key Features & Capabilities
* *Automated Data Extraction:* Successfully parses unstructured financial documents (e.g., PDFs, Word docs) to capture critical transaction details.
* *Intelligent Field Mapping:* Automatically extracts and organizes:
  * *Invoice Details:* Invoice Number (INV-2026-001) and Invoice Date (09 June 2026).
  * *Buyer Details:* Corporate Name (XYZ Enterprises Ltd.), Address, and tax identifiers (GSTIN).
  * *Line-Item Breakdown:* Captures detailed item descriptions, quantities, unit prices, and total amounts (e.g., Office Chairs, Office Desks).
  * *Tax & Totals Calculation:* Automatically processes sub-totals, tax additions (GST @ 18%), and final invoice amounts.
* *Process Efficiency:* Reduces manual processing time from minutes to seconds per document, minimizing data entry lag and errors.

## 🛠️ Tools & Technologies Used
* *Automation Platform:* Nanonets AI Agents
* *Document Formats:* PDF / DOCX (Invoice_to_Purchase_Order.docx)
* *Data Fields Managed:* Invoice Meta-data, Vendor/Buyer Profiles, Itemized Financial Tables
